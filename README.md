# NidusBot
**Desenvolvi esse bot com a intenção de estudar javascript e a biblioteca discord.js.** <br>

Para testar o bot voce precisa instalar a biblioteca **discord.js** <br>
**npm i discord.js** <br>
e precisara do **Dotenv** <br>
**npm i dotenv**
